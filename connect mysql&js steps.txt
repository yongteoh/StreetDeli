1.安装 Node.js
打开官网：https://nodejs.org
下载 LTS 版本 (长期支持版)，比如 20.x。
安装时 记得勾选 "Add to PATH"（这样 node 命令才能全局使用）。
安装完成后，重新打开终端，再试：
node -v 
如果能看到版本号，说明成功了

2.check the js available （use terminal）
cd "C:\User"
node server.js

4.MySQL setting
CREATE DATABASE streetdeli;
USE streetdeli;

CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  email VARCHAR(100) UNIQUE,
  phone VARCHAR(15) UNIQUE,
  password VARCHAR(255) NOT NULL
);
